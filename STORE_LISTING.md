# Chrome Web Store 发布说明

## 扩展信息

### 名称
AI Bilingual Subtitles - 双语字幕翻译

### 简短描述（132 字符）
为在线视频添加 AI 驱动的中英双语字幕，支持 Google 翻译和 OpenAI 兼容接口，预翻译无延迟

### 详细描述

🎬 让外语视频不再难懂！

AI Bilingual Subtitles 是一个强大的 Chrome 扩展，为在线视频实时添加中英双语字幕。采用先进的 AI 翻译技术，让您在观看英文视频时，能够同时看到准确流畅的中文翻译。

✨ 主要功能：

【双翻译服务】
• Google 翻译 - 免费使用，无需配置
• OpenAI 兼容接口 - 支持 GPT-4、DeepSeek、月之暗面等 AI 服务，翻译质量更高

【智能缓存】
• 翻译结果自动缓存，避免重复翻译
• 基于内容哈希的缓存校验，字幕变更自动更新
• 离线查看已缓存的翻译

【流畅体验】
• 全量预翻译，播放时无延迟
• 进度持久化，页面刷新不丢失
• 自动重试机制，网络错误自动恢复

【灵活配置】
• 三种显示模式：中英双语 / 仅中文 / 仅英文
• 可调节批处理大小和请求间隔
• 支持自定义 API 配置

【隐私保护】
• 不收集个人信息
• 数据本地存储
• 开源透明

📖 使用方法：
1. 打开支持的视频网站（如 deeplearning.ai）
2. 播放视频，扩展自动检测字幕
3. 点击扩展图标查看状态和控制面板
4. 高级设置中可配置翻译服务和显示选项

🔧 技术支持：
- 详细文档：https://github.com/yourusername/bilingual-subtitles
- 问题反馈：https://github.com/yourusername/bilingual-subtitles/issues

📋 权限说明：
- activeTab: 检测视频和字幕元素
- storage: 保存翻译缓存和配置
- tabs: 扩展内部通信
- host_permissions: 获取字幕文件和调用翻译 API

⚠️ 注意事项：
- 需要视频网站提供 VTT 格式字幕
- OpenAI 服务需要自备 API Key
- 首次翻译可能需要等待，后续访问自动加载缓存
