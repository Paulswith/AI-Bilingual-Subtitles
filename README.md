# AI 双语字幕 Chrome 扩展

为在线视频添加 AI 驱动的中英双语字幕，支持 Google 翻译和 OpenAI 兼容接口。

## 当前支持的网站

- [x] deeplearning.ai # 初衷也是为了适配它而开发的此插件，谢谢speckit.

其它请提交Issue，也欢迎贡献

## 功能特性

### 🚀 核心功能
- **全量预翻译**：页面加载后自动获取并翻译所有字幕，无等待延迟
- **双翻译服务**：支持 Google 翻译（免费）和 OpenAI 兼容接口
- **智能缓存**：翻译结果本地缓存，避免重复翻译
- **进度持久化**：页面刷新后可恢复翻译进度

### ⚙️ 高级功能
- **重试机制**：网络请求失败时自动指数退避重试
- **缓存校验**：基于字幕内容哈希的缓存键，字幕变更自动重新翻译
- **错误分类**：区分网络错误、API 错误、认证错误等
- **请求限流保护**：自动检测并处理速率限制

### 🎨 显示选项
- 中英双语 / 仅中文 / 仅英文 三种显示模式
- 可调节字幕样式（字体大小、背景、位置等）
- 浮动控制面板，随时调整设置

## 安装方法

### 开发模式安装
1. 下载或克隆本仓库
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目的文件夹

### 从 Chrome Web Store 安装（待发布）
1. 访问 Chrome Web Store
2. 搜索"AI Bilingual Subtitles"
3. 点击"添加到 Chrome"

## 使用说明

### 快速开始
1. 打开支持的视频网站（如 deeplearning.ai）
2. 播放视频，扩展会自动检测字幕
3. 点击扩展图标查看状态和控制面板
4. 翻译结果会自动缓存，下次访问无需重复翻译

### 配置翻译服务

#### Google 翻译（默认）
- 免费使用，无需配置
- 适合日常使用
- 可能会受到速率限制

#### OpenAI 兼容接口
1. 点击扩展图标，打开设置
2. 选择"OpenAI 兼容接口"
3. 配置以下信息：
   - API Base URL（如：https://api.openai.com/v1）
   - API Key
   - 模型名称（如：gpt-3.5-turbo, deepseek-chat 等）
4. 点击"保存配置"

支持的 API 服务商：
- OpenAI API
- Azure OpenAI
- DeepSeek API
- 月之暗面 (Moonshot AI)
- 其他兼容 OpenAI SDK 的服务

### 高级设置
访问 `options.html` 或点击扩展中的"高级设置"：
- 批处理大小：每批翻译的字幕数量
- 请求间隔：批次间的等待时间
- 缓存有效期：翻译缓存保留天数
- 最大重试次数：翻译失败时的重试次数
- 调试日志：开启后在控制台输出详细日志

## 权限说明

| 权限 | 用途 |
|------|------|
| `activeTab` | 读取当前页面信息以检测视频和字幕元素 |
| `storage` | 保存翻译缓存和用户配置到本地存储 |
| `tabs` | 在扩展弹窗和内容脚本之间通信 |
| `host_permissions` | 访问视频网站的字幕文件和翻译服务 API |

## 隐私政策

本扩展：
- **不会**收集个人身份信息或浏览历史
- **不会**将数据发送到除翻译服务外的任何第三方
- 所有配置和缓存仅存储在本地
- API Key 使用 Chrome 安全的 storage API 加密存储

详细隐私政策请参阅 [PRIVACY.md](PRIVACY.md)

## 常见问题

### Q: 翻译失败怎么办？
A: 检查以下几点：
1. 网络连接是否正常
2. 如使用 OpenAI 服务，检查 API Key 是否有效
3. 查看浏览器控制台的错误信息
4. 尝试降低批处理大小或增加请求间隔

### Q: 缓存如何清除？
A: 在扩展弹窗或高级设置中点击"清除缓存"按钮

### Q: 支持哪些网站？
A: 理论上支持任何使用 HTML5 Video 的网站，目前已测试：
- deeplearning.ai
- 其他使用 VTT 字幕的网站

### Q: 可以导出字幕吗？
A: 可以，在弹窗中点击"导出字幕"按钮，将生成 SRT 格式文件
