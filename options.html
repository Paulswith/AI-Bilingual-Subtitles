📖 缓存说明

翻译结果会自动缓存到本地存储，避免重复翻译。

缓存有效期可在高级设置中配置（默认 7 天）。

当翻译配置（服务类型、模型等）变更时，建议清除缓存。

📤 导入/导出配置

导出所有设置和缓存，或从备份文件恢复。

🔒 隐私政策

📊 数据收集说明

本扩展不会收集任何个人身份信息或浏览历史。

字幕内容仅在本地处理，翻译时才会发送至您配置的翻译服务。

使用 Google 翻译时，字幕文本会通过加密连接发送至 Google 服务器。

使用 OpenAI 兼容接口时，字幕文本会发送至您配置的 API 端点。

🔐 数据存储

所有配置和翻译缓存仅存储在您的浏览器本地。

API Key 使用 Chrome 安全的 storage API 加密存储。

您可以随时清除所有数据（见缓存管理标签页）。

🌐 网络请求

扩展仅在以下情况发起网络请求：

1. 获取视频字幕文件（来自视频网站）

2. 调用翻译服务 API（您主动选择的服务）

3. 检查扩展更新（由 Chrome 浏览器管理）
