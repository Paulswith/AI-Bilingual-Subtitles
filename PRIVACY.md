# 隐私政策

**AI 双语字幕扩展**（以下简称"本扩展"）尊重并保护您的隐私。本隐私政策说明我们如何收集、使用和保护您的信息。

## 最后更新日期

2024年1月

## 信息收集

### 我们收集的信息

本扩展**不会**主动收集以下信息：
- 个人身份信息（姓名、邮箱、电话等）
- 浏览历史记录
- 视频观看记录
- 设备标识符
- 位置信息

### 本地存储的数据

本扩展仅在您的浏览器本地存储以下数据：
- **翻译缓存**：已翻译的字幕内容，用于避免重复翻译
- **用户配置**：API 设置、显示偏好等扩展配置
- **API Key**（可选）：仅在您配置 OpenAI 兼容服务时存储

### 网络传输的数据

本扩展仅在以下情况发送数据到外部服务器：

1. **获取字幕文件**：从视频网站获取原始字幕内容
2. **翻译请求**：将字幕文本发送至您选择的翻译服务
   - Google 翻译：发送至 Google 翻译 API
   - OpenAI 兼容接口：发送至您配置的 API 端点

## 数据安全

### 本地存储安全

- 所有配置数据使用 Chrome 浏览器提供的安全 storage API 存储
- 数据仅保存在您的设备上，不会同步到我们的服务器
- 您可以随时通过扩展的"缓存管理"功能清除所有数据

### 网络传输安全

- 所有 API 请求均通过 HTTPS 加密传输
- API Key 仅用于您配置的翻译服务，不会被发送到其他服务器
- 我们不存储、记录或分析您的翻译内容

## 第三方服务

### Google 翻译

当您选择使用 Google 翻译时：
- 字幕文本通过加密连接发送至 Google 翻译 API
- 请查阅 [Google 隐私政策](https://policies.google.com/privacy) 了解 Google 如何处理数据

### OpenAI 兼容接口

当您选择使用 OpenAI 兼容接口时：
- 字幕文本发送至您指定的 API 端点
- 我们不对第三方 API 提供商的数据处理负责
- 请查阅您选择的 API 提供商的隐私政策

## 您的权利

您拥有以下权利：
- **查看数据**：通过浏览器开发者工具查看存储的数据
- **导出数据**：使用扩展的导出功能备份您的配置
- **删除数据**：使用"清除缓存"功能删除所有存储的数据
- **禁用扩展**：随时在 Chrome 扩展管理中禁用或卸载本扩展

## 儿童隐私

本扩展不面向 13 岁以下儿童。我们不会有意收集儿童的个人信息。

我们可能会不时更新本隐私政策。更新后的政策将在扩展商店页面和 GitHub 仓库中公布。

## 权限说明

本扩展需要以下权限才能正常工作：

| 权限 | 用途 |
|------|------|
| `activeTab` | 读取当前页面以检测视频和字幕元素 |
| `storage` | 保存翻译缓存和用户配置 |
| `tabs` | 扩展弹窗与内容脚本之间的通信 |
| `host_permissions` | 获取视频网站字幕文件和调用翻译 API |
